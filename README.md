# piLynk

Share files, folders and text instantly from your Android phone over your local Wi-Fi network.

---

## Features

- 📂 Folder Sharing
- 📄 File Sharing
- 📝 Text Sharing
- 📷 QR Code Generation
- 🌐 Built-in HTTP Server
- ⚡ No Internet Required
- 🔒 Data never leaves your device
